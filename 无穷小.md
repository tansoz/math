# 无穷小

## 判断无穷小

$ \lim_{ x \to x_0 } \dfrac{ f(x) }{ g(x) } = 0 $ 高阶无穷小。

$ \lim_{ x \to x_0 } \dfrac{ f(x) }{ g(x) } = C $ 同阶无穷小。当 $ C = 1 $ 时，称为等价无穷小。

$ \lim_{ x \to x_0 } \dfrac{ f(x) }{ g(x) } = \infty $ 低阶无穷小。

## 判断是几阶无穷小

$ \lim_{ x \to x0 } \dfrac{ f(x) }{ g(x)^k } $ 表示 $ f(x) $ 是 $ g(x) $ 的 k 阶无穷小。
